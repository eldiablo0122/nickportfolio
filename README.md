# Nicole Abella Portfolio Website

## How to View

Double-click `start-website.bat`, then open:

http://127.0.0.1:5173

If the page is already running, refresh the browser.

## Important

Do not open `index.html` directly from File Explorer. This is a Vite React website, so it needs the local dev server.

## Files

- `src/main.jsx` - website content and React components
- `src/styles.css` - premium responsive styling
- `dist/` - production build generated by `npm run build`
